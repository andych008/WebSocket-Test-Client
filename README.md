# log-client

show remote log. work with [log-server](https://github.com/andych008/log-server)

logcat --> log-server --> log-client 


## run
```
npm install
npm run start
```

## install in chrome
1. 下载我已经打包好的crx  https://pan.baidu.com/s/1lRqIK_9S0US2giw5AxUQCg
1. 解压log-client.crx
2. 安装 `chrome --> extensions --> Load unpacked`
